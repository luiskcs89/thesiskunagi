Summary

Ilarkesto ist a toolkit which contains usefull java-classes, java-frameworks and shell-scripts.

Documentation

The only documentation is the javadoc.

Some Classes

  • Str - Utilitiy methods for working with text/strings. Cutting, comparing, parsing, modifying.
  • IO - Utility methods for working with files, streams and images. Reading, copying, converting.
  • Sys - Utilitiy methods for the java core. System properties access.
  • Tm - Utilitiy methods for dealing with date and time. Current month, year. Date comparsions.
  • Utl - Utility methods. Randomization, array and collection conversions.
  • Money - Working with money (amount + currency).
  • Env - Encapuslation of platform specific operations. Executing programs and files. System directories.
  • Logger - Simple, minimal and performant logging system.
  • Swing - Utility methods for Swing. Dialogs, frames, positioning.
  • JDom - Utility methods for reading and writing XML with JDom.

Some Frameworks

  • Date & and; Time - Easy classes for gregorian date, time and periods. Comparing, converting, formatting, parsing.
  • Persistence - Creating persistent objects (entities).
  • Functional Programming - Interfaces for functions and predicates. Implementation for different operations based on functions or predicates.
  • Model Driven Architecture - Models and generators for entities and components.
  • Project Building - Classes for building projects. Compiling, packaging, deploying. Convention over configuration.
